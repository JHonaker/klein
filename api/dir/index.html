    Direction

    group dir

    Directions in \(\mathbf{P}(\mathbb{R}^3_{3, 0, 1})\) are represented using points at infinity (homogeneous coordinate 0). Having a homogeneous coordinate of zero ensures that directions are translation-invariant.
